# virus_pipeline
Viral de novo assembly pipeline.

This repo contains the pipeline for de novo viral assembly pipeline. It was developed at University College London under the ICONIC* project and is largely the work of Dr Dan Frampton and Matthew Byott. The pipeline was developed under publicly funded research and is provided open-source for academic use.

## Requirements
-Samtools
-SMALT
-Trimmomatic
-IVA (Docker version preferable)
-R (ggplot2)
